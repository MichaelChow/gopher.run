# 4.3 容器和集合

1. **内置容器**
```go
/usr/local/go/src/
├── container/          # 容器类型
│   ├── heap/           # 堆实现
│   │   └── heap.go     # 堆操作
│   ├── list/           # 链表
│   │   └── list.go     # 链表操作
│   └── ring/           # 环形链表
│       └── ring.go     # 环形链表操作
└── 其他容器实现
```

1. **排序和搜索**
```go
/usr/local/go/src/
├── sort/               # 排序算法
│   ├── search.go       # 二分搜索
│   ├── sort.go         # 排序实现
│   └── zfuncversion.go # 函数版本
└── 其他算法实现
```
